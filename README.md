Metro Ticket Management System (SQL Project)

Project Overview

This project demonstrates a Metro Ticket Management System built using SQL. 
The system manages information related to metro stations, passengers, ticket bookings, and travel records. 
It helps organize transportation data and allows efficient querying of metro ticket operations.

Project Objective

The objective of this project is to practice database design and
SQL queries for managing a transportation ticketing system and retrieving useful information from stored data.

Technologies Used

SQL
Database Management Concepts

Key Features

Store passenger information
Manage metro stations and routes
Record ticket booking details
Retrieve travel data using SQL queries
Analyze metro transportation records

Database Operations Used
The project demonstrates the use of SQL commands such as:
SELECT
INSERT
UPDATE
DELETE
JOIN
GROUP BY
ORDER BY


Learning Outcomes
Through this project, the following concepts were practiced:

Database design
Writing SQL queries
Data retrieval and filtering
Data management using relational databases

Project Purpose
This project was created as part of learning SQL and database management to understand how ticket booking systems work in real-world transportation systems.
